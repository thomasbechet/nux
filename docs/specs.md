# Console Specifications

NUX console supports multiple specs based on the cartridge options.

### Frame Rate

| Name       | Value      |
| ---------- | ---------- |
| Resolution | 512x320    |
| FPS        | 60         |
| Memory     | 64Mb       |

## Color Palette

- 1 colormap : 256 RGB colors
- 1 palette  : 256 indices

## Textures

- palette textures : index in palette
- colored textures : full RGB colors

## Framebuffers

- Full RGB colors
