# Console Specifications

NUX console supports multiple specs based on the cartridge options.

## Graphics

### Resolution

| Name | Resolution |
| ---- | ---------- |
| Low  | 128x128    |

### Frame Rate

| Name | Frame rate |
| ---- | ---------- |
| Low  | 30         |

### GPU Resources

| Name     | Memory Available |
| -------- | ---------------- |
| Textures | 16Mb             |
| Vertices | 32Mb             |
